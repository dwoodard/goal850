# Project Summary

## Project Description
Establish a Credit Rebuilding Company dedicated to providing a wide array of financial services designed to help both individuals enhance their credit scores. Our services will include digitally automated credit counseling, debt management strategies, and tailored financial education programs, all aimed at empowering our clients to achieve better creditworthiness and financial stability. 

## Goals 
- **Goal 1**: Create a visually appealing and intuitive blogging platform powered by Statamic, featuring easy navigation, engaging layouts, and responsive design.
  - **1.1**: Statimic CMS: Develop a custom blogging platform using Statamic CMS.
    - **1.1.1**: Create custom templates and layouts for the blog.
    - **1.1.2**: Implement dynamic content management for blog posts.
  - **1.2**: Develop an admin panel for content management.
  - **1.3**: Conduct SEO audits and implement performance enhancements.

- **Goal 2**: Create a way to add Podcasts with episodes.
  - **2.1**: Design the podcast submission form.
  - **2.2**: Build a system for episode uploads.
  - **2.3**: Integrate a podcast player on the site.

- **Goal 3**: Add Array.com API to give users access to podcast content.
  - **3.1**: we need to add array components, HTML Embed of Credit, Identity Protection, Privacy Protection, Subscription Management


## Roadmap (Current Phase: Phase 1)
- **Phase 1**: By the end of Q4, we'll have a fully functional blogging platform and a way to add Podcasts with episodes.
- **Phase 2**: Add Marketing Go Highlevel, focusing on automated marketing techniques.
- **Phase 3**: Launch a mobile app.


## Technologies
- **Tailwind**: Utilized for styling and layout, ensuring a sleek and modern look and feel.
- **Statamic**: Used for developing the blogging platform, providing a modern and intuitive interface.
- **Laravel**: The underlying PHP framework that supports application development, ensuring robust and scalable solutions.
- **Array.com API**: Integrated to provide users with access to podcast content, enhancing the platform's multimedia capabilities.
- **Pest**: A PHP testing framework to ensure code quality and reliability through automated testing.
- **Laravel Forge**: Utilized for deploying the application to the cloud, streamlining the deployment process.

## Site Map
- **Home**: Landing page with a brief overview.
- **Blog**: Collection of blog posts.
- **Podcasts**: Collection of podcasts.
- **About**: Information about the company.
- **Contact**: Form for users to contact the company.
